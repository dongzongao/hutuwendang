# 锁概览

## 锁的全景图

```
锁
├── 数据库锁（MySQL）
│   ├── 按粒度
│   │   ├── 表锁
│   │   │   ├── 读锁 / 写锁
│   │   │   └── MDL 元数据锁
│   │   └── 行锁（InnoDB）
│   │       ├── 记录锁（Record Lock）
│   │       ├── 间隙锁（Gap Lock）
│   │       └── 临键锁（Next-Key Lock）
│   ├── 按模式
│   │   ├── 共享锁 S（读锁）
│   │   └── 排他锁 X（写锁）
│   ├── 意向锁（IS / IX）
│   └── 按思想
│       ├── 悲观锁
│       └── 乐观锁（version / CAS）
│
├── Java 并发锁
│   ├── synchronized（监视器锁）
│   │   └── 锁升级：偏向锁 → 轻量级锁 → 重量级锁
│   ├── ReentrantLock（可重入锁）
│   │   ├── 公平锁 / 非公平锁
│   │   └── 基于 AQS
│   ├── ReadWriteLock（读写锁）
│   ├── StampedLock（乐观读锁）
│   ├── Semaphore（信号量）
│   ├── CountDownLatch / CyclicBarrier
│   └── CAS（无锁）
│
└── 分布式锁
    ├── Redis 分布式锁（SETNX / Redisson）
    ├── ZooKeeper 分布式锁（临时顺序节点）
    └── 数据库分布式锁（唯一索引）
```
