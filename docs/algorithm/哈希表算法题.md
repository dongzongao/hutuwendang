# 哈希表算法题

> 核心思想：用空间换时间，O(1) 查找代替 O(n) 遍历。

---

## 1. 两数之和

遍历时把每个数存入 HashMap，同时查找 `target - 当前值` 是否已存在，一次遍历 O(n)。

## 2. 字母异位词分组

## 3. 最长连续序列

先把所有数存入 Set，只从序列起点（`num-1` 不在 Set 中）开始向右扩展，O(n)。
