# 简单算法题与解题思路

## 1. 两数之和

思路：暴力解是双层循环 O(n²)。优化用 HashMap，遍历时把每个数存进去，同时查找 `target - 当前值` 是否已存在，一次遍历搞定。

```java
public int[] twoSum(int[] nums, int target) {
    Map<Integer, Integer> map = new HashMap<>();
    for (int i = 0; i < nums.length; i++) {
        int complement = target - nums[i];
        if (map.containsKey(complement)) {
            return new int[]{map.get(complement), i};
        }
        map.put(nums[i], i);
    }
    return new int[]{};
}
// 时间 O(n)，空间 O(n)
```

## 2. 反转链表

思路：用三个指针 `prev / curr / next`。每次让 `curr.next` 指向 `prev`，然后三个指针整体向右移动一位，直到 `curr` 为 null，`prev` 就是新头节点。

```
原始: 1 → 2 → 3 → null
步骤: null ← 1   2 → 3
      null ← 1 ← 2   3
      null ← 1 ← 2 ← 3
```

```java
public ListNode reverseList(ListNode head) {
    ListNode prev = null, curr = head;
    while (curr != null) {
        ListNode next = curr.next; // 先保存下一个
        curr.next = prev;          // 反转指针
        prev = curr;               // prev 前进
        curr = next;               // curr 前进
    }
    return prev;
}
// 时间 O(n)，空间 O(1)
```

## 3. 有效括号

思路：遇到左括号就入栈，遇到右括号就弹出栈顶检查是否匹配。最后栈为空说明全部匹配。关键边界：右括号时栈为空直接返回 false。

```
"([{}])"
遇到 ( → 栈: [( ]
遇到 [ → 栈: [(, []
遇到 { → 栈: [(, [, {]
遇到 } → 弹出 { 匹配 ✓ 栈: [(, []
遇到 ] → 弹出 [ 匹配 ✓ 栈: [( ]
遇到 ) → 弹出 ( 匹配 ✓ 栈: []
栈空 → true
```

```java
public boolean isValid(String s) {
    Deque<Character> stack = new ArrayDeque<>();
    for (char c : s.toCharArray()) {
        if (c == '(' || c == '[' || c == '{') {
            stack.push(c);
        } else {
            if (stack.isEmpty()) return false;
            char top = stack.pop();
            if (c == ')' && top != '(') return false;
            if (c == ']' && top != '[') return false;
            if (c == '}' && top != '{') return false;
        }
    }
    return stack.isEmpty();
}
```

## 4. 爬楼梯

思路：到第 n 阶只能从第 n-1 或 n-2 阶跳过来，所以 `f(n) = f(n-1) + f(n-2)`，就是斐波那契数列。不需要数组，滚动两个变量就行，空间压到 O(1)。

```
n=1: 1种  [1]
n=2: 2种  [1+1, 2]
n=3: 3种  [1+1+1, 1+2, 2+1]
n=4: 5种  = f(3)+f(2) = 3+2
```

```java
public int climbStairs(int n) {
    if (n <= 2) return n;
    int a = 1, b = 2;
    for (int i = 3; i <= n; i++) {
        int c = a + b;
        a = b;
        b = c;
    }
    return b;
}
// 时间 O(n)，空间 O(1)
```

## 5. 二分查找

思路：数组有序，每次取中间值和目标比较，目标更大就去右半段，更小就去左半段，每次排除一半，直到找到或区间为空。注意 `mid = left + (right - left) / 2` 防止两数相加溢出 int。

```
nums = [1,3,5,7,9], target = 7

left=0, right=4, mid=2 → nums[2]=5 < 7 → left=3
left=3, right=4, mid=3 → nums[3]=7 == 7 → return 3
```

```java
public int search(int[] nums, int target) {
    int left = 0, right = nums.length - 1;
    while (left <= right) {
        int mid = left + (right - left) / 2;
        if (nums[mid] == target) return mid;
        else if (nums[mid] < target) left = mid + 1;
        else right = mid - 1;
    }
    return -1;
}
// 时间 O(log n)，空间 O(1)
```

## 6. 滑动窗口最大值

思路：用双端队列（Deque）维护一个单调递减队列，队头始终是当前窗口最大值。新元素入队前，把队尾所有比它小的元素弹出，保证队列单调性。

```
nums=[1,3,-1,-3,5,3,6,7], k=3

窗口[1,3,-1] → 队列[3,-1]  → 最大3
窗口[3,-1,-3] → 队列[3,-1,-3] → 最大3
窗口[-1,-3,5] → 队列[5]    → 最大5
窗口[-3,5,3]  → 队列[5,3]  → 最大5
```

```java
public int[] maxSlidingWindow(int[] nums, int k) {
    int[] result = new int[nums.length - k + 1];
    Deque<Integer> deque = new ArrayDeque<>(); // 存下标

    for (int i = 0; i < nums.length; i++) {
        // 移除超出窗口的队头
        if (!deque.isEmpty() && deque.peekFirst() < i - k + 1) {
            deque.pollFirst();
        }
        // 维护单调递减，弹出比当前小的队尾
        while (!deque.isEmpty() && nums[deque.peekLast()] < nums[i]) {
            deque.pollLast();
        }
        deque.offerLast(i);
        // 窗口形成后记录结果
        if (i >= k - 1) {
            result[i - k + 1] = nums[deque.peekFirst()];
        }
    }
    return result;
}
// 时间 O(n)，空间 O(k)
```

## 7. 最长无重复子串（滑动窗口）

思路：用左右双指针维护一个窗口，用 HashMap 记录字符最新出现的位置。右指针右移时，若字符已在窗口内，左指针跳到该字符上次出现位置的下一位，全程记录最大窗口长度。

```
s = "abcabcbb"

right=0: 窗口[a]       len=1
right=1: 窗口[ab]      len=2
right=2: 窗口[abc]     len=3
right=3: a重复，left跳到1，窗口[bca] len=3
right=4: b重复，left跳到2，窗口[cab] len=3
```

```java
public int lengthOfLongestSubstring(String s) {
    Map<Character, Integer> map = new HashMap<>();
    int max = 0, left = 0;

    for (int right = 0; right < s.length(); right++) {
        char c = s.charAt(right);
        if (map.containsKey(c) && map.get(c) >= left) {
            left = map.get(c) + 1; // 左指针跳过重复字符
        }
        map.put(c, right);
        max = Math.max(max, right - left + 1);
    }
    return max;
}
// 时间 O(n)，空间 O(n)
```

## 8. 合并两个有序链表

思路：双指针分别指向两个链表头，每次取较小的节点接到结果链表，直到某个链表遍历完，把剩余部分直接接上。

```
l1: 1→2→4
l2: 1→3→4

比较: 1==1取l1的1 → 比较1和1取l2的1 → 比较2和3取2 → ...
结果: 1→1→2→3→4→4
```

```java
public ListNode mergeTwoLists(ListNode l1, ListNode l2) {
    ListNode dummy = new ListNode(0); // 哑节点简化边界处理
    ListNode curr = dummy;

    while (l1 != null && l2 != null) {
        if (l1.val <= l2.val) {
            curr.next = l1;
            l1 = l1.next;
        } else {
            curr.next = l2;
            l2 = l2.next;
        }
        curr = curr.next;
    }
    curr.next = (l1 != null) ? l1 : l2; // 接上剩余部分
    return dummy.next;
}
// 时间 O(m+n)，空间 O(1)
```

## 9. 最大子数组和（动态规划）

思路：`dp[i]` 表示以 `nums[i]` 结尾的最大子数组和。状态转移：要么把当前元素加到前面的子数组，要么从当前元素重新开始，取两者较大值。用滚动变量优化空间。

```
nums = [-2,1,-3,4,-1,2,1,-5,4]

dp:    -2  1  -2  4   3  5  6   1  5
                   ↑ 从4重新开始（前面dp=-2<0，加上没意义）
最大值 = 6
```

```java
public int maxSubArray(int[] nums) {
    int maxSum = nums[0], curr = nums[0];

    for (int i = 1; i < nums.length; i++) {
        curr = Math.max(nums[i], curr + nums[i]); // 要么续上，要么重新开始
        maxSum = Math.max(maxSum, curr);
    }
    return maxSum;
}
// 时间 O(n)，空间 O(1)
```

## 10. 判断回文串

思路：双指针从两端向中间夹，逐一比较字符是否相等，遇到非字母数字字符跳过。

```
"A man, a plan, a canal: Panama"
去掉非字母数字 → "amanaplanacanalpanama"
双指针对比 → 全部匹配 → true
```

```java
public boolean isPalindrome(String s) {
    int left = 0, right = s.length() - 1;
    while (left < right) {
        while (left < right && !Character.isLetterOrDigit(s.charAt(left))) left++;
        while (left < right && !Character.isLetterOrDigit(s.charAt(right))) right--;
        if (Character.toLowerCase(s.charAt(left)) != Character.toLowerCase(s.charAt(right))) {
            return false;
        }
        left++;
        right--;
    }
    return true;
}
// 时间 O(n)，空间 O(1)
```

## 11. 三数之和

思路：先排序，固定一个数 `nums[i]`，剩下两个用双指针夹逼。排序后可以方便跳过重复元素，避免结果重复。

```
nums = [-4,-1,-1,0,1,2]

固定 i=-4: left=-1, right=2 → -4+-1+2=-3<0, left右移
固定 i=-1: left=0, right=2  → -1+0+2=1>0, right左移
           left=-1, right=1 → -1+-1+1=-1<0 ...
           找到 [-1,-1,2], [-1,0,1]
```

```java
public List<List<Integer>> threeSum(int[] nums) {
    Arrays.sort(nums);
    List<List<Integer>> result = new ArrayList<>();

    for (int i = 0; i < nums.length - 2; i++) {
        if (i > 0 && nums[i] == nums[i-1]) continue; // 跳过重复
        int left = i + 1, right = nums.length - 1;
        while (left < right) {
            int sum = nums[i] + nums[left] + nums[right];
            if (sum == 0) {
                result.add(List.of(nums[i], nums[left], nums[right]));
                while (left < right && nums[left] == nums[left+1]) left++;
                while (left < right && nums[right] == nums[right-1]) right--;
                left++; right--;
            } else if (sum < 0) {
                left++;
            } else {
                right--;
            }
        }
    }
    return result;
}
// 时间 O(n²)，空间 O(1)
```

## 12. 接雨水

思路：对每个位置，能接的水 = `min(左边最高, 右边最高) - 当前高度`。预处理左右最大值数组，然后遍历累加。

```
height = [0,1,0,2,1,0,1,3,2,1,2,1]

位置2: min(左max=1, 右max=3) - height[2]=0 → 接水1
位置4: min(左max=2, 右max=3) - height[4]=1 → 接水1
位置5: min(左max=2, 右max=3) - height[5]=0 → 接水2
...总计 6
```

```java
public int trap(int[] height) {
    int n = height.length;
    int[] leftMax = new int[n], rightMax = new int[n];

    leftMax[0] = height[0];
    for (int i = 1; i < n; i++)
        leftMax[i] = Math.max(leftMax[i-1], height[i]);

    rightMax[n-1] = height[n-1];
    for (int i = n-2; i >= 0; i--)
        rightMax[i] = Math.max(rightMax[i+1], height[i]);

    int water = 0;
    for (int i = 0; i < n; i++)
        water += Math.min(leftMax[i], rightMax[i]) - height[i];

    return water;
}
// 时间 O(n)，空间 O(n)
```

## 13. 二叉树层序遍历（BFS）

思路：用队列，每次把当前层所有节点出队，同时把它们的子节点入队，记录每层结果。

```
        3
       / \
      9  20
        /  \
       15   7

层序结果: [[3],[9,20],[15,7]]
```

```java
public List<List<Integer>> levelOrder(TreeNode root) {
    List<List<Integer>> result = new ArrayList<>();
    if (root == null) return result;

    Queue<TreeNode> queue = new LinkedList<>();
    queue.offer(root);

    while (!queue.isEmpty()) {
        int size = queue.size(); // 当前层节点数
        List<Integer> level = new ArrayList<>();
        for (int i = 0; i < size; i++) {
            TreeNode node = queue.poll();
            level.add(node.val);
            if (node.left != null) queue.offer(node.left);
            if (node.right != null) queue.offer(node.right);
        }
        result.add(level);
    }
    return result;
}
// 时间 O(n)，空间 O(n)
```

## 14. 最小栈

思路：用两个栈，一个正常存数据，一个同步维护当前最小值。每次 push 时，辅助栈压入当前最小值；pop 时两个栈同步弹出。

```
操作: push(-2), push(0), push(-3), getMin, pop, getMin
主栈:  [-2, 0, -3]
辅栈:  [-2,-2,-3]  ← 每次记录当前最小
getMin → 辅栈顶 = -3
pop后辅栈 → [-2,-2], getMin = -2
```

```java
class MinStack {
    Deque<Integer> stack = new ArrayDeque<>();
    Deque<Integer> minStack = new ArrayDeque<>();

    public void push(int val) {
        stack.push(val);
        int min = minStack.isEmpty() ? val : Math.min(val, minStack.peek());
        minStack.push(min);
    }

    public void pop() {
        stack.pop();
        minStack.pop();
    }

    public int top() { return stack.peek(); }

    public int getMin() { return minStack.peek(); }
}
// 所有操作 O(1)
```

## 15. 买卖股票最佳时机

思路：一次遍历，记录历史最低价，每天计算"今天卖出"的利润，取最大值。

```
prices = [7,1,5,3,6,4]

day1: min=7, profit=0
day2: min=1, profit=0
day3: min=1, profit=5-1=4
day4: min=1, profit=3-1=2
day5: min=1, profit=6-1=5 ← 最大
day6: min=1, profit=4-1=3
最大利润 = 5
```

```java
public int maxProfit(int[] prices) {
    int minPrice = Integer.MAX_VALUE, maxProfit = 0;
    for (int price : prices) {
        minPrice = Math.min(minPrice, price);
        maxProfit = Math.max(maxProfit, price - minPrice);
    }
    return maxProfit;
}
// 时间 O(n)，空间 O(1)
```

## 16. 快速排序

思路：选一个基准值（pivot），把数组分成左边（小于pivot）和右边（大于pivot）两部分，然后递归对两部分排序。平均 O(n log n)，最坏 O(n²)（已排序数组）。

```
[3,6,8,10,1,2,1]  pivot=3

分区后: [1,2,1] 3 [6,8,10]
递归左: [1,1,2]
递归右: [6,8,10]
结果:   [1,1,2,3,6,8,10]
```

```java
public void quickSort(int[] nums, int left, int right) {
    if (left >= right) return;
    int pivot = partition(nums, left, right);
    quickSort(nums, left, pivot - 1);
    quickSort(nums, pivot + 1, right);
}

private int partition(int[] nums, int left, int right) {
    int pivot = nums[right]; // 选最右为基准
    int i = left - 1;
    for (int j = left; j < right; j++) {
        if (nums[j] <= pivot) {
            i++;
            int tmp = nums[i]; nums[i] = nums[j]; nums[j] = tmp;
        }
    }
    // 把 pivot 放到正确位置
    int tmp = nums[i+1]; nums[i+1] = nums[right]; nums[right] = tmp;
    return i + 1;
}

// 调用
quickSort(nums, 0, nums.length - 1);
// 时间 O(n log n) 平均，O(n²) 最坏，空间 O(log n) 递归栈
```

## 17. 归并排序

思路：分治思想，把数组不断二分直到只剩一个元素，然后两两有序合并。稳定排序，时间复杂度稳定 O(n log n)，但需要额外 O(n) 空间。

```
[8,4,5,7,1,3,6,2]

分:  [8,4,5,7]      [1,3,6,2]
分:  [8,4] [5,7]    [1,3] [6,2]
分:  [8][4] [5][7]  [1][3] [6][2]
合:  [4,8] [5,7]    [1,3] [2,6]
合:  [4,5,7,8]      [1,2,3,6]
合:  [1,2,3,4,5,6,7,8]
```

```java
public void mergeSort(int[] nums, int left, int right) {
    if (left >= right) return;
    int mid = left + (right - left) / 2;
    mergeSort(nums, left, mid);
    mergeSort(nums, mid + 1, right);
    merge(nums, left, mid, right);
}

private void merge(int[] nums, int left, int mid, int right) {
    int[] tmp = new int[right - left + 1];
    int i = left, j = mid + 1, k = 0;

    while (i <= mid && j <= right) {
        if (nums[i] <= nums[j]) tmp[k++] = nums[i++];
        else tmp[k++] = nums[j++];
    }
    while (i <= mid) tmp[k++] = nums[i++];
    while (j <= right) tmp[k++] = nums[j++];

    // 拷贝回原数组
    for (int p = 0; p < tmp.length; p++) {
        nums[left + p] = tmp[p];
    }
}

// 调用
mergeSort(nums, 0, nums.length - 1);
// 时间 O(n log n)，空间 O(n)，稳定排序
```
