# 动态规划算法题

> 核心思想：把大问题拆成子问题，用状态转移方程递推，避免重复计算。三要素：状态定义、转移方程、初始值。

---

## 1. 爬楼梯

`f(n) = f(n-1) + f(n-2)`，斐波那契数列，滚动变量压缩空间到 O(1)。

## 2. 最大子数组和（Kadane 算法）

`dp[i] = max(nums[i], dp[i-1] + nums[i])`，前缀和为负就重新开始。
