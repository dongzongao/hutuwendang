# 算法题学习路径

按难度和依赖关系分阶段，每个阶段打牢再进入下一阶段。

---

## 第一阶段：基础数据结构（2周）

先搞懂结构，再做题。

```
数组 → 链表 → 栈/队列 → 哈希表
```

必刷题：
- 数组：两数之和、移除元素、合并有序数组
- 链表：反转链表、合并两个有序链表、判断环形链表
- 栈：有效括号、最小栈、用栈实现队列
- 哈希：两数之和、字母异位词分组、最长连续序列

---

## 第二阶段：双指针 + 滑动窗口（1周）

解决数组/字符串的区间问题，模板固定，上手快。

```
双指针：左右夹逼（三数之和、接雨水）
滑动窗口：左右扩缩（最长无重复子串、最小覆盖子串）
```

必刷题：
- 三数之和
- 最长无重复子串
- 接雨水
- 滑动窗口最大值
- 最小覆盖子串

---

## 第三阶段：二分查找（1周）

有序 = 想二分，模板背熟。

```java
// 通用模板
int left = 0, right = n - 1;
while (left <= right) {
    int mid = left + (right - left) / 2;
    if (check(mid)) right = mid - 1;
    else left = mid + 1;
}
```

必刷题：
- 二分查找（基础）
- 搜索旋转排序数组
- 寻找峰值
- 在排序数组中查找元素的第一个和最后一个位置

---

## 第四阶段：树（2周）

二叉树是高频考点，DFS/BFS 两种遍历方式必须熟练。

```
前序/中序/后序（DFS递归）→ 层序遍历（BFS）→ BST 特性
```

必刷题：
- 二叉树的三种遍历（递归 + 迭代）
- 层序遍历
- 二叉树最大深度 / 最小深度
- 对称二叉树
- 路径总和
- 二叉搜索树验证
- 最近公共祖先

---

## 第五阶段：动态规划（3周）

最难也最重要，从一维到二维，从简单到复杂。

```
一维DP → 二维DP → 背包问题 → 区间DP → 状态压缩DP
```

学习顺序：
```
爬楼梯（入门）
→ 打家劫舍（一维）
→ 最大子数组和
→ 买卖股票系列
→ 最长递增子序列（LIS）
→ 最长公共子序列（LCS）
→ 0-1背包
→ 编辑距离
→ 正则表达式匹配（进阶）
```

---

## 第六阶段：图（2周）

```
DFS/BFS 遍历 → 拓扑排序 → 最短路径 → 并查集
```

必刷题：
- 岛屿数量（DFS/BFS）
- 课程表（拓扑排序）
- 腐烂的橘子（多源BFS）
- 冗余连接（并查集）

---

## 第七阶段：回溯（1周）

排列组合子集类问题，模板固定。

```java
// 回溯模板
void backtrack(参数) {
    if (终止条件) { 收集结果; return; }
    for (选择 : 选择列表) {
        做选择;
        backtrack(参数);
        撤销选择;
    }
}
```

必刷题：
- 全排列
- 子集
- 组合总和
- 电话号码的字母组合
- N皇后

---

## 刷题平台和节奏建议

平台：LeetCode（主）、牛客（面试真题）

节奏：
```
每天 2-3 题，不要贪多
一道题想超过30分钟没思路 → 直接看题解，理解后自己写一遍
同类型题连续刷，形成肌肉记忆
每周复盘错题
```

面试前重点：
```
高频考点：动态规划、二叉树、双指针、滑动窗口
必须手写：快排、归并排序、二分查找、反转链表
```
