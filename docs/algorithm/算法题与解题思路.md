# 算法题与解题思路

按题型分类，每道题附带可交互的代码运行器，点击「▶ 运行」后展示逐步执行流程。

## 分类导航

| 分类 | 核心思想 | 代表题目 |
|------|---------|---------|
| [哈希表](./哈希表算法题) | 空间换时间，O(1) 查找 | 两数之和、字母异位词分组、最长连续序列 |
| [双指针](./双指针算法题) | 利用有序性消除嵌套循环 | 三数之和、接雨水、最长无重复子串 |
| [动态规划](./动态规划算法题) | 状态转移，避免重复计算 | 爬楼梯、最大子数组和、零钱兑换 |
| [栈与队列](./栈与队列算法题) | LIFO/FIFO 处理"最近"关系 | 有效括号、最小栈、滑动窗口最大值 |
| [树与BFS](./树与BFS算法题) | 递归分解子问题，队列层序 | 层序遍历、最大深度、路径总和 |
| [排序](./排序算法题) | 分治思想，稳定性与复杂度权衡 | 快速排序、归并排序 |

## 解题框架

```
1. 明确输入输出 → 边界条件（空数组、单元素、负数）
2. 暴力解 → 分析瓶颈 → 针对性优化
3. 常见优化手段：
   - O(n²) 查找 → HashMap O(1)
   - 嵌套循环 → 双指针 / 滑动窗口
   - 重复子问题 → 动态规划 / 记忆化
   - 最近元素关系 → 单调栈
   - 图/树遍历 → BFS(队列) / DFS(递归)
```
