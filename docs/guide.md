# 文档说明

## 简介

本文档整理了 Java 后端开发的核心技术点，内容涵盖 Java 基础、JVM、Spring 生态、数据库、缓存、中间件、网络协议、安全加密、架构设计和算法等方向。

适合有一定开发经验的后端工程师系统梳理知识体系，也可作为日常开发的参考手册。

---

## 文档结构

| 分类 | 内容 |
|------|------|
| Java | 基础语法、线程并发、Stream API、正则表达式、数据安全 |
| JVM | 内存模型、垃圾回收器（G1/ZGC）、类加载机制 |
| Spring 生态 | SpringBoot 原理、IOC 容器、WebFlux、Reactor |
| 数据库 | MySQL 原理、事务、索引与回表 |
| 缓存 | Redis 原理、数据结构、源码解读 |
| 中间件 & 微服务 | Nacos、ZooKeeper、服务治理、Gateway、MQ |
| 网络 & 协议 | HTTP/HTTPS/HTTP2、TCP/UDP、多路复用、通信协议 |
| 安全 & 加密 | 对称加密、非对称加密、TLS 原理 |
| 架构设计 | 幂等设计 |
| 算法 | 学习路径、常见题型解析、复杂度分析 |
