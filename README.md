# 后端技术文档

> Java 后端开发技术文档，涵盖 JVM、Spring、数据库、中间件、网络、算法等方向，持续更新。

## 在线访问

[https://github.com/dongzongao/hutuwendang](https://github.com/dongzongao/hutuwendang)

## 本地运行

```bash
npm install
npm run dev
```

访问 http://localhost:5173

## 文档目录

| 分类 | 内容 |
|------|------|
| Java | 基础、线程并发、Stream、正则、数据安全 |
| JVM | 内存模型、G1/ZGC 垃圾回收器、类加载机制 |
| Spring 生态 | SpringBoot、IOC 容器、WebFlux、Reactor |
| 数据库 | MySQL、事务、索引回表 |
| 缓存 | Redis 原理与源码解读 |
| 中间件 & 微服务 | Nacos、ZooKeeper、服务治理、Gateway、MQ |
| 网络 & 协议 | HTTP/HTTPS/HTTP2、TCP/UDP、多路复用 |
| 安全 & 加密 | 对称加密与非对称加密 |
| 架构设计 | 幂等设计 |
| 算法 | 学习路径、常见题解、复杂度分析 |
